# ✨ Character Counter

A simple and responsive **Character Counter** web application built using **HTML, CSS, and JavaScript**. It allows users to type a message and displays the number of characters entered in real time while enforcing a maximum character limit.


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## ✨ Features

- 📝 Real-time character counting
- 🔢 Displays current and maximum character count
- 🚫 Maximum character limit (200 characters)
- 📱 Responsive design for desktop and mobile
- 🎨 Clean and modern user interface
- ⚡ Fast and lightweight

## 💡 How It Works

1. Enter your message in the text area.
2. The character count updates instantly as you type.
3. Users can type up to **200 characters**.
4. The application prevents users from exceeding the maximum limit.
